# Error Tracker

A Flutter plugin to handle unhandled exceptions.

## Getting Started

**How to install**

* Go to your project's `pubspec.yaml` file.
* Add `errortracker` plugin in your project dependencies.
  ```
  errortracker:
      git:
        url: https://github.com/samarjeetr2001/error-tracker.git
        ref: master

  ```
